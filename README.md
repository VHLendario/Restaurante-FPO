# Restaurante – FPO

Este projeto implementa um sistema simples de gerenciamento para um restaurante utilizando **Fila**, **Pilha** e **Lista Encadeada**, aplicadas a situações reais do dia a dia de um estabelecimento.

# Fila de Pedidos
Gerencia os pedidos feitos pelos clientes:
- Adicionar pedido  
- Entregar pedido (FIFO)  
- Listar pedidos pendentes  

# Pilha de Pratos
Controla os pratos sujos chegando à cozinha:
- Receber prato sujo (PUSH)  
- Lavar prato (POP)  
- Listar pilha de pratos  

# Lista de Insumos
Gerencia ingredientes e insumos:
- Adicionar insumo  
- Remover insumo por posição  
- Listar insumos
